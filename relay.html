OpenAPI request generation

OpenAPI 3.x and Swagger 2.0 JSON are imported with operations, parameters, examples, request bodies, tags, and server URLs. Common YAML layouts are also recognized.

Tip: JSON preserves the most detail. YAML import focuses on server and operation discovery without an external parser dependency.
